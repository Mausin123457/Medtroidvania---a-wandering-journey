# Metroidvania Journey
My journey to my first ever metroidvania game.

============================================================================

I have always wanted to create a Metroidvania game after playing hollow knight. I technically once made a game before in gdevelop when I was 14, but I am not going to count that as an actual game.
I am going to be using godot to create this game and update this repo frequently. This will be my third time using godot (I have tried to use it twice before, but i quickly gave up or lost interest). 
This time I am more motivated then ever to create this game and I am using git/github as a way to not lose motivation on creating this. I will update this README in the future for some devlogs or interesting progress that I have made. All of my commits will also have a short description of what I was working on + what I will be working on for the next commit, so those will also kind of double as a more thorough devlog.

I don't yet know what the full name of the game or the lore of this game will be, but it will be fleshed out as i go and create characters, enemies, levels and bosses. Everything in this project will be made by me. I do not intent to use any AI or other peoples work for this game. All the assets, music, visuals will be made by hand, by me! Now that all of the boring stuff has been written down, i will update this README in the future for devlogs, or other interesting messages I may have!
